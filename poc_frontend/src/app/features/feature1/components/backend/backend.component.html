Backend integration

In order to integrate with the backend, add a proxy.conf.json file where you define how pattern of routes invoked by the frontend relate to the backend:
{{'{ "/api": { "target": "http://localhost:8080", "pathRewrite":{"^/api" : ""} } } '}}

This configuration can be used either by modifying package.json to :
"start": "ng serve --proxy-config proxy.conf.json"
and starting the server with npm start

or by starting the server with ng serve --proxy-config proxy.conf.json
